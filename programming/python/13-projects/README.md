# 13 — Projects

## Overview

Apply everything you've learned to build real programs. Each project is self-contained in its own directory with a README, main entry point, and optional requirements.

## Projects

### 1. Todo CLI
A command-line task manager with file persistence. Practice: file I/O, data structures, argument parsing.

### 2. Text Adventure
An interactive text-based game. Practice: classes, game loops, state management.

### 3. Web Scraper
Scrape data from a public website. Practice: HTTP requests, HTML parsing, data extraction.

### 4. API Client
Consume a REST API (weather or GitHub). Practice: HTTP requests, JSON parsing, error handling.

## Getting Started

```bash
cd todo-cli
python3 main.py --help

cd text-adventure
python3 main.py

# For projects with dependencies:
cd web-scraper
pip install -r requirements.txt
python3 main.py
```
