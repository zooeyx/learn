# learn

Structured learning: C programming, AI/ML, and more.

## Structure

```
programming/c/    — C from scratch (chapters 01-11)
ai-ml/            — AI/ML (placeholder)
```

## Quick Start

```bash
cd programming/c/01-basics
make          # build all exercises
make run      # build + run
make ASAN=1   # with AddressSanitizer
```
